# Valiuchenko Vladislav's Portfolio website
It's just a minimalistic personal portfolio website

## Technologies used
- [TypeScript](https://www.typescriptlang.org) — strong typization over JavaScript
- [Preact](https://preactjs.com) — UI library allowing usage of JSX
- [SASS](https://sass-lang.com) — CSS preprocessor
- [Webpack](https://webpack.js.org) — app bundler
